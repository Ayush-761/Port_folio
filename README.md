# Ayush Raj - Personal Portfolio Website

A premium portfolio website showcasing my journey as a BTech student in data analysis and web development. Features a visual-first design with strong sections, minimal text, and professional presentation.

## 🌐 Live Demo

**View Live**: https://ayush-761.github.io/portfolio

*Deployed using GitHub Pages for simple hosting.*

## ⭐ Key Features

- **Visual-First Design**: Bigger visuals, fewer words, stronger sections
- **Premium Animations**: Glow effects, smooth transitions, and hover interactions
- **Responsive Layout**: Optimized for desktop, tablet, and mobile devices
- **Glassmorphism UI**: Modern backdrop blur effects and gradient styling
- **Technology Badges**: Clean skill presentation with visual badges
- **Featured Projects**: Centerpiece project cards with enhanced visuals

## 🎨 Design Highlights

### Hero Section - Simplified Impact
- Clean tagline: "Building Data-Driven Solutions"
- Glow card with key statistics
- Minimal text, maximum visual impact

### Projects Section - Centerpiece Focus
- Single featured Power BI dashboard project
- Large visual card with glow borders
- Professional presentation with tech badges

### Skills Section - Technology Badges
- Core technologies in visual badge format
- Currently learning section with distinct styling
- Clean, scannable layout
